# **Congruence**
---
## Setup

Clone repository:
```git clone```

Install dependencies:
```npm install```

## Run
#### Start testrpc (with 0x) and server
```npm start```

#### Start testrpc (with 0x) only
```npm run testrpc```

*__Note__: testrpc default port is 8545*

#### Start server only
```npm run server```

*__Note__: server default port is 8000*
